# 🏃‍♂️ AI Tag Simulator: Genetic Evolution & Strategy

자바스크립트로 구현된 **지능형 AI 술래잡기 시뮬레이션**입니다. 
단순한 이동을 넘어 Steering Behaviors, 유전 알고리즘, 상태 머신을 결합하여 시간이 지날수록 진화하는 개체들의 생존 사투를 보여줍니다.

## 🚀 기술 스택 (Tech Stack)
* **Language:** JavaScript (ES6+)
* **Graphics:** HTML5 Canvas API (2D)
* **Styling:** CSS3 (UI & Animation)
* **Environment:** Web Browser (Chrome, Edge 등)

---

## 🧬 핵심 알고리즘 (Main Algorithms)

### 1. Steering Behaviors (조향 행동)
* **Seek (추격):** 타겟의 위치를 예측하여 가속합니다.
* **Flee (도망):** 술래의 위치로부터 반대 방향으로 도망칩니다.
* **Obstacle Avoidance:** **Ray-casting** 센서를 이용해 보이지 않는 레이저를 쏴서 앞의 벽을 감지하고 회피합니다.

### 2. Genetic Algorithm (유전 및 진화)
* **DNA 상속:** 도망자 전멸 시, 가장 오래 살아남은 상위 5마리의 DNA(속도, 시야 등)를 복제합니다.
* **Mutation (변이):** 다음 세대는 부모의 DNA에 미세한 변이를 일으켜 더 빠르고 영리하게 태어납니다.

### 3. Flocking (군집 제어)
* **Separation (분산):** 도망자들이 서로 겹치지 않게 적절한 거리를 유지하며 심리적 안정감을 얻습니다.

## 👹 술래 조건 (Chaser Logic) - 2025_12_31 Update

* **지능형 타겟팅:** 살아있는 도망자 중 가장 가까운 개체를 우선적으로 추격합니다.
* **성장 시스템 (XP):** 도망자를 잡을 때마다 **Level Up**하며, 레벨당 속도가 **0.05**씩 영구 상승합니다.
* **피로도 (Fatigue):** 추격 시간이 길어질수록 속도가 저하되며, 포획 성공 시 초기화됩니다.
* **데드락 방지:** 장애물에 끼일 경우(40프레임 이상) 경로를 재탐색하여 탈출합니다.
* **특수 상태:**
    * **리스폰:** 인원 부족 시 **Lv.10** 상태로 강력하게 부활합니다.
    * **광폭화 (Berserk):** 최후의 1인 생존 시 발동. 속도가 **7.5**로 대폭 상승하며 바리케이드를 파괴합니다.
    * **약점:** 중앙 **Mud Zone** 진입 시 속도가 평소의 **25%**로 급감합니다.

## 🏃‍♂️ 도망자 조건 (Evader Logic)

* **생존 리소스:** 초당 에너지가 소모되며, 0이 되면 굶어 죽습니다.
* **대시 (Dash):** 술래가 80px 이내로 접근하면 에너지를 써서 속도를 **1.8배** 높입니다.
* **군집 보너스:** 동료와 모여 있으면 속도가 최대 **2.5**만큼 추가 상승합니다.
* **전략적 협력:** * **술래 사냥:** 3마리 이상이 술래 근처(80px)에 모이면 술래를 제거할 수 있습니다.
    * **동료 부활:** 전체 먹은 음식 합계가 12개가 될 때마다 죽은 동료를 부활시킵니다.
* **방어 기제:** * **바리케이드:** 술래 접근 시 갈색 벽을 설치합니다. (Mud Zone 내 설치 불가)
    * **은신 (Hide):** 장애물 뒤로 숨어 술래의 시야를 피합니다.
