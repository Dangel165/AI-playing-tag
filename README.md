# AI-playing-tag
자바스크립트로 만든 AI 술래잡기입니다 


JavaScript:술래와 도망자의 인공지능(AI), 물리 연산, 게임 루프 등 핵심 로직을 담당합니다.
HTML5 Canvas:2D 그래픽 출력을 위한 도화지 역할을 합니다
CSS3:UI 디자인과 애니메이션 을 담당합니다.
Web Browser:별도의 설치 없이 크롬, 엣지 등의 브라우저에서 즉시 실행됩니다

적용된 주요 알고리즘
Steering Behaviors (조향 행동):Seek(추격): 목표를 향해 가속,Flee(도망): 술래로부터 도망,Avoid(회피): 장애물을 감지하고 옆으로 꺾음

Flocking (군집 제어):도망자들이 서로 겹치지 않게 밀어내는 Separation(분산) 로직이 적용되었습니다

Ray-casting (레이캐스팅):술래가 보이지 않는 레이저를 쏴서 앞의 벽을 미리 감지하는 지능형 센서 기술입니다

Genetic Algorithm (유전 알고리즘):도망자가 죽으면 우수한 개체(오래 살아남은 자)의 능력치(dna)를 다음 세대에 물려주고 변이(mutate)를 일으키는 진화 로직입니다
